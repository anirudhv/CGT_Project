App Description - Purdue Mobile Menus

With all the great meal options at Purdue's dining courts, choosing where to eat can be tough. With Purdue Mobile Menus, it just got a lot easier.

With a simple-to-use interface, Purdue mobile Menus puts the following informatoin at your fingertips:

  • Daily menus for each dining court
  • Vegetarian options
  • Dining court hours
  • Contact information
  • Maps

"Everyday before breakfast, lunch, and dinner, this app is extremely convenient in seeing what each dining court is serving so that I can go eat what I want to have." - Jamal Hansen (Purdue Class of 2021)
